RB Post Views Widget plugin use for your posts visit count.

== Description ==
RB Post View Widget is a modern and easy to change plugin. With this plugin you can easily see the most visiting posts of any type of post types.

== Installation ==

### From your WordPress dashboard

= Visit =
Plugins > Add New

= Search =
for "RB Post Views Widget"

= Install and Activate =
"RB Post Views Widget" from your Plugins page

= Go to =
Appearance -> Widgets -> Widgets List: Widgets Category -> RB Post Views Widget

= Active Widget =
Actiated your widget and save.
